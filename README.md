# Canteen Menu Rating System

## Overview

The Canteen Menu Rating System is a web application that allows users to view daily canteen menus, rate food items, and provide feedback. It helps canteen management understand customer preferences and improve food quality.

## Features

- View daily canteen menu
- Rate food items
- Submit feedback and reviews
- Display average ratings
- User-friendly interface

## Technologies Used

- Java
- Spring Boot
- Maven
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Configure the MySQL database.
4. Run the Spring Boot application.
5. Open the application in your browser.
